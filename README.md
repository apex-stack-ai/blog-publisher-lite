# Blog Publisher Lite

Free Claude skill for cross-publishing blog articles to Dev.to via API.

## Features
- Publish articles to Dev.to using the REST API
- Set tags, canonical URLs, and series
- Draft or publish directly

## Installation
Download the `.skill` file and install it in Claude Desktop (Cowork mode).
